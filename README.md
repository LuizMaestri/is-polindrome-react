# Is Polindrome

#
## Backend

Install dependencies
<pre>
    npm install
</pre>
or
<pre>
    yarn
</pre>

run server
<pre>
    npm start
</pre>
or
<pre>
    yarn start
</pre>

## Frontend

Install dependencies
<pre>
    npm install
</pre>
or
<pre>
    yarn
</pre>

run server dev mode
<pre>
    npm server-dev
</pre>
or
<pre>
    yarn server-dev
</pre>

run server
<pre>
    npm start
</pre>
or
<pre>
    yarn start
</pre>
